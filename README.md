MoodMate is a lightweight and beginner-friendly mood tracking application I built using Python and MySQL/SQLite.
Its purpose is simple: let users log their mood each day, view trends, and understand how their emotions change over time.

I made this project to learn how apps collect, store, and visualize personal data, and how a basic digital wellness tool works behind the scenes.

Features
1. Daily Mood Logging

Users can select or enter their mood for the day:

Happy

Neutral

Sad

Stressed

Custom moods

2. View Mood History

Shows all previous entries with:

Date

Mood

Optional notes

3. Mood Statistics

Based on recorded data, MoodMate shows:

Most frequent mood

Total number of logs

Streak of consistent entries

Weekly/Monthly summary (if implemented)

4. Database Integration

Uses a SQL database to store:

User moods

Dates

Optional comments

Supports MySQL or SQLite depending on your setup.
